## Folder Structure

Each exercise should be contained in its own folder using the naming convention of `concept#-name-of-concept`. See example below:

```
exercises
    |_ concept1-hello-world
    |   |_ starter
    |   |   |_ exercise-files
    |   |_ solution
    |   |   |_ solution_files
    |_ concept2-foo-bar
    |   |_ starter
    |   |   |_ exercise-files
    |   |_ solution
    |   |   |_ solution_files
    |_ README.md
```
